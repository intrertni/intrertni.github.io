解析Epub电子书记录

（一）浏览Epub文件

0.epub格式介绍

EPUB（全称为Electroinc Publication）格式内部采用了XHTML标准，该格式的文件内容支持根据阅读设备的特性以最适合浏览的方式显示。

epub文件本质是就是个压缩文件。

解压后的目录结构如图：

进入text目录，如下图，里面都是html格式的文件。本质上，epub就是包装了网页。那么通过网页就可以展示书籍了。

即，利用epub解析库将epub文件解析成多个网页文件，就能通过webView浏览文件内容。

注意目录结构中的content.opf文件，打开可以看到这个电子书epub资源的组织形式。

可以看到代码里的id，每个id对应一个资源文件。

1.具体解析步骤如下：
01.配置环境

  1. 首先，到[epub官网](Epublib – a java epub library – Paul Siegmann)下载最新的epub解析库，即，epublib-core-latest.jar,以及它的依赖库slf4j-api和slf4j-android这两个jar包。

  2. 然后，将这三个jar包放到App模块的lib目录下。

  3. 最后，在Android Studio 工作目录project结构下，选中这三个jar包，右键Add as Library..，之后AS会把jar包引入App模块。

  • 或者直接在build.gradle(app)里配置依赖库:

02 .准备所需的epub文件

将epub文件，放到App模块的src\main\assets目录下，准备将其解析成一系列HTML文件。

03.将epub文件保存在手机存储卡上

在java代码中打开epub文件，将其引用的所有资源文件（包括网页、图片等）都保存到存储卡上，同时获取该电子书所有页面对应的HTML路径列表，以便后续的界面展示操作。
